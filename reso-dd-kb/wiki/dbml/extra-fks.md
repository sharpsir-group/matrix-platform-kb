# Extra FKs (prose + name-shape detection)

This is the human-readable companion to [`reso-2.0-canonical.dbml`](./reso-2.0-canonical.dbml). It enumerates every foreign-key relationship in the canonical schema that was discovered by something other than a Resource-typed sibling row in `raw/fields.csv` (the primary FK signal).

Three signals (in priority order):

1. **`prose-foreign-key`** - Definition contains *"foreign key relating to the X Resource"*.
2. **`prose-resources`** - Definition contains *"X Resource's YKey"*.
3. **`name-shape`** - Field name `<Word>Key` where `<Word>` is itself a Resource name (or its singular form for plural Resources like `Contacts`/`Teams`/`Rules`).

**32 extra FKs** are emitted from these signals (after filtering hits that match satellite-dropped columns).

Generated by [`scripts/build_reso_canonical_dbml.py`](../../scripts/build_reso_canonical_dbml.py). Do not edit by hand.
