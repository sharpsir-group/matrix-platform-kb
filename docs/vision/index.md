## Key Concepts

- **Sharp Matrix Platform**: Multi-app ecosystem with RESO DD 2.0 as canonical data layer
- **3 Markets**: Cyprus (core), Hungary (growth), Kazakhstan (emerging luxury)
- **4 Client Segments**: UHNWI (€2M+), HNWI (€1-2M), Standard (€300K-1M), Mass (<€300K)
- **4 Elements of the Model**: Commission Funnel, Listings, Appointments, Follow-ups
- **2 Representations**: Manager Kanban + AI Brokerage Copilot for brokers
- **AI Copilot**: Context analysis → Next Best Action → Automated routines
- **Dual Pipeline**: Seller-side (listings) + Buyer-side (purchases) running in sync
- **Matching Engine**: AI connects listings ↔ buyers via Curated Lists
- **Trigger Boundary**: First meeting/showing = transition from Follow-up to Active Sales
- **7-Phase Roadmap**: Jan 2026 → Feb 2027, from foundation to full integration
- **Target**: 90%+ automation, 50-day sales cycle, €115K client LTV
