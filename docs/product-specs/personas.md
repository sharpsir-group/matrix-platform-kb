# User Personas

> **For Lovable**: These personas define who uses Matrix Apps.
> Use these when making UX decisions.

---

## 1. Elena K. — Luxury Broker

| Attribute | Value |
|-----------|-------|
| **Role** | Luxury Broker (Cyprus) |
| **Goals** | Manage 15–20 active listings, mobile access, minimal data entry |
| **Pain Points** | Manual data entry, switching between systems |
| **Key Metrics** | Listings managed, showings scheduled, conversion rate |
| **Apps Used** | Broker App, Client Portal, Marketing Platform |
